# Instituto Gomes Basquete

Site institucional em Next.js para o Instituto Gomes Basquete, projeto sem fins lucrativos com basquete gratuito e atividades comunitárias para crianças e adolescentes.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- lucide-react
- framer-motion

## Como rodar

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
npm run lint
npm run build
```

## Assets
